This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Instructions to use:

Follow this steps:

### Clone this repo:

`https://github.com/epicnessinha/epicnessinha.github.io-movieland.git`

### Install npm package:

`npm install`

### Run the app:

`npm start`
